HTML Basics:


HTML, Hyper Text Markup Language is comprised of:
  • Text content - ie what you see.
  • Markup - the construction blocks of html that define what the text looks like.
  • References to other documents (Links to video, and images).
  • Links to other pages.
Overview of HTML Markup:

HTML is comprised of tags which defines the behavior and appearance of a webpage. Tags are structured as follows:

< name >content </name>.
  • < name > - is called the opening tag.
  • contents - is called the tag contents.
  • < /name > - is called the closing tag.

The set of tags is also referred too as an Element. It is important to remember that the behavior of a tag will
remain in effect until it is closed </>! Example elements found in this exercise
  • < b >Will make text bold </b >
  • < em >Will make text italic < /em >
  • < a ></a > is an anchor tag. An example of this tag in use is in the creation of a link to another webpage. That tag is constructed with the following syntax: <a href = "url"> URL Plain text </a>. In use it would look something like this:

    G+ Community Homepage!
  • <img src = "URL" alt = "alternate text"> Will embed an image here.
Note: A closing tag is not needed for this element. It is whats referred to as a Void Tag.
Notes on Whitespace


There are two types of whitespace
  1. In-line
  2. Block
In-line Whitespace: Ends a line by wrapping text to a new line.

Examples of in-line elements are:
  • a
  • span
  • br
  • img
  • strong

Block Whitespace: Adds an invisible box around the text encapsulated by the tag

Examples of in-line elements are:
  • p
  • div
  • form


The Structure of an HTML Document:


The bare bones markup of an HTML document looks like this...

<!DOCTYPE HTML>
<html>
      < head >
            <title> TITLE </title>
      < /head>
      < body>

      </body>
</html>

Lesson 2: Creating a Structured HTML Document


Front End web development Overview

Front end web development is visual. A good conceptual construct of a website is like a house. There are three
components of this house:

  • HTML: Acts as the structure like where the walls are placed or the location of the room
  • CSS: Acts as the style of the house like the color and texture of the walls or the type of decorations
  • JavaScript: Acts as the interactive components of a house like a remote control for the TV or a garage door opener

Web browsers have development tools that allow the user to view source code/styling and interactively (and temporarily)modify code on the fly.

Document Object Model (DOM)

The Document Object Model is a tree like structure of a web-page. It is a cross platform language convention for representing and interacting with objects. The nodes of every document are organized in a tree structure called a DOM Tree. In HTML these objects are known as elements.


Boxes, Grids, and Rules


The architecture of a website is setup as a series of boxes that are aligned with a grid. This allows the
developer to re-arrange the web-page to change the user experience of the site with relative ease.

Lesson 3: Adding CSS Style to HTML Structure


Cascading Style Sheets

Cascading means that the rules are applied no only to the elements they directly match
but also to all those child elements. However, if the child element has multiple overlapping
rules defined for it, the more specific rule takes effect.

CSS Styling Laws

Styles can be defined in different places and are applied in the following order with definitions
further down the list overwriting previous definitions. The more specific rule is applied to every element.
A great example of how the styling laws work are how geographic and political boundaries are set up in the United States:

  • National
  • State
  • County
  • City
  • Residence
Controlling HTML with CSS

In CSS, the element you wish to control is referred to as the selector. Each line within the selector is called a
declaration. The text to the left of the ":" is the property - and the text to the right is called the value.
For example:

    p {
        color:white
        }

Will set the color of all paragraph text to white.

Class selectors apply styles to specific classes within the respective name defined in the HTML.
using .class-name

You can include CSS in a web page by:
  • Write CSS in the head of HTML
  • Link your HTML to a separate CSS File (this is the preferred method)
  • Write the styling in-line with your HTML (this is not ideal...at all)
Code, Test, Refine (repeat)

There are a few steps to follow to become better at coding:

  1. Look for Natural Blocks in web pages
  2. Look for repeated styles and syntax elements
  3. Write HTML code
  4. Apply Styles (from big to small)
  5. Fix Things
Developer tools in browsers such as Google Chrome allow programmers to make changes to code on the fly
